PiPo API

Detailed Description

The PiPo API is composed of :

  • the core Plugin API (PiPo.h, PiPoSequence.h, PiPoParallel.h and RingBuffer.h)
  • the Host API which is a superset of the Plugin API, containing all the files listed above.
